# 关于简单圣经

## 项目简介

简单圣经是一个基于 [Nextra](https://nextra.site) 框架构建的现代化圣经阅读平台。我们的目标是提供一个简洁、快速、易用的在线圣经阅读体验。

## 技术栈

- **Next.js** - React 框架
- **Nextra** - 文档站点生成器
- **React** - UI 库
- **Tailwind CSS** - 样式框架

## 特点

### 🚀 性能优化
基于 Next.js 的静态站点生成，提供极快的加载速度。

### 📱 响应式设计
完美适配桌面、平板和手机等各种设备。

### 🔍 全文搜索
内置强大的搜索功能，快速找到您需要的经文。

### 🌓 深色模式
支持深色和浅色主题切换，保护您的眼睛。

### ♿ 无障碍访问
遵循 Web 无障碍标准，让每个人都能轻松使用。
